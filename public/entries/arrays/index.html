    Overview#

    Arrays are one of the simplest data structures.

    Core Idea#

    Elements are stored in contiguous memory locations.

    Complexity Analysis#

    • Access: \( O(1) \)
    • Search: O(n)

    Code snippet#

    int arr[5] = {1,2,3,4,5};
    int arr[] = {1, 2, 3};
    arr = [1, 2, 3, 4, 5]
    let arr : [i32; 5] = [1, 2, 3, 4, 5];
