## About this book

This is a community-driven, encyclopedic reference for
Data Structures, Algorithms, and related computer science topics.

Unlike tutorial sites, entries here are designed to be:
- Concise
- Correct
- Language-agnostic
- Backed by reference implementations

## How to use this book

- Use the **left sidebar** to browse topics
- Use the **right sidebar** to navigate subtopics within a page
- Use the **code language selector** to switch implementations globally

## Scope and philosophy

This book prioritizes:
- Conceptual clarity
- Visual explanations where possible
- Reference-quality code over competitive programming tricks

## How to contribute

Contributions are welcome.
Each entry follows a fixed schema:
- Overview
- Core Idea
- Complexity Analysis
- Reference Code
